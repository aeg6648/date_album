💕 편집 페이지

사진/후기 추가하고 데이트 수정하려면 GitHub 토큰이 필요해요
토큰 만드는 법 (한 번만):
1. 이 링크 클릭
2. Token name: "Date Album Editor" (아무거나)
3. Expiration: 원하는 기간 (1년 등)
4. Repository access → Only select repositories → aeg6648/date_album
5. Permissions → Repository permissions → Contents를 Read and write로
6. Generate token → 나온 토큰 복사해서 위에 붙여넣기

1. console.anthropic.com/settings/keys 접속
2. Create Key 클릭 → 이름 아무거나 → 복사
3. 위 칸에 붙여넣기

⚠️ 무료 크레딧도 주고, 데이트 하나 생성에 약 1~2원 정도라 저렴해요.
